# Property

## File format

The table data must have <code>class</code> property whose value is "PropertyData".

The <code>type</code> property defines member type.
The <code>value<code> property defines member values.

<pre>
{
  // object type
  class: "Properties"

  // member type
  type: {
    field1:   "n"
    field2:   "b"
  }

  // member value
  value: {
    field1: 123
    field2: true
  }
}
</pre>
